# 用户分群逻辑

## 分群维度

### 1. 理赔价值维度

**分群标准**：基于理赔频率和理赔金额

| 分群 | 特征 |
|------|------|
| 高频高额 | 理赔频率高且理赔金额大 |
| 低频低额 | 理赔频率低且理赔金额小 |

**计算方法**：
1. 选择特征：`claim_count`, `total_claim_amount`, `claim_frequency`, `claim_severity`
2. 标准化特征
3. 使用K-means聚类算法分为2类
4. 根据聚类中心确定高低价值分群

### 2. 用户状态维度

**分群标准**：基于用户活跃度和忠诚度

| 分群 | 特征 |
|------|------|
| 深度忠诚 | 保单数多、保费高、注册时间长、最近有理赔 |
| 流失风险 | 保单数少、保费低、注册时间短、长期无理赔 |

**计算方法**：
1. 选择特征：`days_since_last_claim`, `policy_count`, `total_premium`, `days_since_registration`
2. 标准化特征
3. 使用K-means聚类算法分为2类
4. 根据忠诚度评分确定用户状态分群

## 组合分群

**组合分群结果**：

| 理赔价值 | 用户状态 | 组合分群 |
|----------|----------|----------|
| 高频高额 | 深度忠诚 | 高价值忠诚用户 |
| 高频高额 | 流失风险 | 高价值流失风险 |
| 低频低额 | 深度忠诚 | 低价值忠诚用户 |
| 低频低额 | 流失风险 | 低价值流失风险 |

## 分群特征分析

### 高价值忠诚用户
- **特征**：理赔频率高、理赔金额大、保单数多、保费高
- **策略建议**：提供专属增值服务，引导加保
- **目标**：提高二开率，增加保费收入

### 高价值流失风险
- **特征**：理赔频率高、理赔金额大、但活跃度下降
- **策略建议**：个性化挽回方案，提高留存
- **目标**：降低流失率，维持高价值

### 低价值忠诚用户
- **特征**：理赔频率低、理赔金额小，但忠诚度高
- **策略建议**：培养保险意识，逐步提升保额
- **目标**：提升保费，培养潜在高价值用户

### 低价值流失风险
- **特征**：理赔频率低、理赔金额小、活跃度低
- **策略建议**：基础保障教育，引导转化
- **目标**：提高转化漏斗转化率

## 分群更新频率

建议每月更新一次分群结果，确保分群的及时性和准确性。

## 分群应用场景

1. **精准营销**：针对不同分群制定差异化营销策略
2. **实验设计**：在高潜力分群中进行A/B测试
3. **资源分配**：优先向高价值分群倾斜资源
4. **风险控制**：对高风险分群进行重点监控
