# 用户分群文档

## 1. 分群目标

基于用户的价值、忠诚度和风险特征，将用户划分为不同的群体，为差异化运营策略提供依据。

## 2. 分群维度

### 2.1 价值维度
- **理赔金额**: 用户历史理赔金额总和
- **保费贡献**: 用户历史保费总和
- **保单数量**: 用户持有的保单数量

### 2.2 忠诚度维度
- **续保率**: 历史续保保单数 / 历史到期保单数
- **保单活跃度**: 有效保单数 / 总保单数
- **客户 tenure**: 注册至今的天数

### 2.3 风险维度
- **理赔频率**: 历史理赔次数
- **投诉风险**: 历史投诉次数
- **欺诈风险**: 历史欺诈标记

## 3. 分群逻辑

### 3.1 分位数计算
- 理赔金额分位数：PERCENT_RANK() OVER (ORDER BY total_claim_amount)
- 保费分位数：PERCENT_RANK() OVER (ORDER BY total_premium)
- 续保率分位数：PERCENT_RANK() OVER (ORDER BY renewal_rate)

### 3.2 分群规则

| 分群 | 条件 |
|------|------|
| 高价值忠诚用户 | 理赔金额分位数 >= 0.6 AND 保费分位数 >= 0.6 AND 续保率分位数 >= 0.6 |
| 高价值流失风险 | 理赔金额分位数 >= 0.6 AND 保费分位数 < 0.6 AND 续保率分位数 < 0.6 |
| 低价值忠诚用户 | 理赔金额分位数 < 0.6 AND 续保率分位数 >= 0.6 |
| 低价值流失风险 | 理赔金额分位数 < 0.6 AND 续保率分位数 < 0.6 |

## 4. 分群特征

### 4.1 高价值忠诚用户
- **特征**: 理赔金额高、保费高、续保率高
- **占比**: 约22%
- **策略**: 专属服务 + 加保引导
- **预期效果**: 二开率 +1.5ppt

### 4.2 高价值流失风险
- **特征**: 理赔金额高、保费低、续保率低
- **占比**: 约26%
- **策略**: 个性化挽回 + 续保优惠
- **预期效果**: 续保率 +2.0ppt

### 4.3 低价值忠诚用户
- **特征**: 理赔金额低、保费低、续保率高
- **占比**: 约25%
- **策略**: 培育计划 + 保额提升
- **预期效果**: 保费 +10%

### 4.4 低价值流失风险
- **特征**: 理赔金额低、保费低、续保率低
- **占比**: 约26%
- **策略**: 基础保障教育
- **预期效果**: 转化率提升

## 5. 分群应用

### 5.1 策略匹配

根据用户分群结果，匹配不同的运营策略：

| 分群 | 策略类型 | 优先级 |
|------|----------|--------|
| 高价值忠诚用户 | 加保引导 | 1 |
| 高价值流失风险 | 挽回策略 | 2 |
| 低价值忠诚用户 | 培育计划 | 3 |
| 低价值流失风险 | 基础运营 | 4 |

### 5.2 实验设计

针对高价值用户（高价值忠诚用户 + 高价值流失风险）进行A/B测试：
- 实验组：实施差异化运营策略
- 对照组：不实施特殊策略

### 5.3 效果评估

通过以下指标评估策略效果：
- 续保率提升
- 二开率提升
- 月新增保费增长
- ROI（投入产出比）
