# 实验设计指南

## 实验目标

通过A/B测试验证理赔场景下的运营策略效果，以提高保费收入和用户满意度。

## 实验设计原则

### 1. 因果推断为核心
- 所有策略效果必须通过随机对照实验验证
- 确保实验结果的统计显著性
- 提供效应量和置信区间

### 2. 用户分层实验
- 支持按核心用户分层进行分组分析
- 识别策略对不同人群的异质处理效应
- 针对不同分群制定差异化策略

### 3. 归因分析
- 明确保费增量归因逻辑
- 仅归因实验期内由策略直接触发的加保/复购订单
- 排除其他因素的干扰

## 实验设计步骤

### 1. 确定实验目标
- 明确要测试的策略
- 定义核心指标和护栏指标
- 设定实验期限

### 2. 样本量计算
- **输入参数**：
  - 基础转化率
  - 最小可检测效应
  - 显著性水平（默认0.05）
  - 功效（默认0.8）
- **计算方法**：
  ```python
  sample_size = calculate_sample_size(
      baseline_conversion=0.05,
      minimum_detectable_effect=0.15
  )
  ```

### 3. 随机分流
- **分流单位**：用户级
- **分流比例**：默认50%对照组，50%测试组
- **分层分流**：确保各分群在两组中的分布均衡

### 4. 实验监控
- **实时监控**：核心指标和护栏指标
- **预警机制**：当指标异常时及时预警
- **SRM检验**：确保分组均衡性

### 5. 统计分析
- **转化率检验**：卡方检验
- **收入检验**：t检验
- **置信区间**：95%置信区间
- **效应量**：Cohen's d或其他效应量指标

### 6. 结果评估
- **主要指标**：
  - 二开率提升
  - 续保率提升
  - 增量保费
- **次要指标**：
  - 用户满意度
  - 理赔时效
  - 投诉率

### 7. 决策与推广
- **决策标准**：
  - 主指标显著改善
  - 护栏指标无恶化
  - 投资回报率为正
- **推广策略**：
  - 从小规模试点开始
  - 逐步扩大覆盖范围
  - 持续优化策略

## 实验类型

### 1. 策略测试
- **测试内容**：不同理赔场景下的营销策略
- **示例**：理赔完成后推荐不同类型的保险产品

### 2. 时机测试
- **测试内容**：不同理赔阶段的触达时机
- **示例**：理赔前、理赔中、理赔后的不同触达时机

### 3. 渠道测试
- **测试内容**：不同触达渠道的效果
- **示例**：短信、APP推送、电话回访等

### 4. 内容测试
- **测试内容**：不同营销内容的效果
- **示例**：不同的文案、优惠方案等

## 实验注意事项

1. **实验污染**：避免测试组和对照组之间的信息泄露
2. **样本流失**：监控实验期间的用户流失情况
3. **季节性影响**：考虑季节因素对实验结果的影响
4. **长期效应**：关注策略的长期效果，而非短期收益
5. **道德考量**：确保实验设计符合伦理要求
