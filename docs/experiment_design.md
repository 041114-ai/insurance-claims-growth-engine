# 实验设计文档

## 1. 实验目标

验证针对不同用户分群的差异化运营策略对续保率和二次开发率的影响。

## 2. 实验设计

### 2.1 实验类型
- **类型**: 用户级分层随机实验
- **随机化单位**: 用户
- **实验周期**: 30天（短期）+ 90天（长期）

### 2.2 实验分组
- **对照组**: 不实施任何特殊策略
- **测试组**: 实施差异化运营策略
  - 高价值流失风险用户：个性化挽回 + 续保优惠
  - 高价值忠诚用户：专属服务 + 加保引导

### 2.3 样本量计算

使用以下参数计算所需样本量：
- 基准转化率: 20%
- 最小可检测效应: 15%
- 显著性水平: 0.05
- 统计功效: 0.8

计算结果：每组需要约500个用户。

## 3. 实验指标

### 3.1 主指标
| 指标 | 定义 | 计算方法 |
|------|------|----------|
| 续保率 | 保单到期后续保的比例 | 续保保单数 / 到期保单数 |
| 二开率 | 现有客户购买新保单的比例 | 二开保单数 / 活跃客户数 |

### 3.2 护栏指标
| 指标 | 定义 | 阈值 |
|------|------|------|
| 投诉率 | 理赔客户投诉比例 | 不超过基线的110% |
| 理赔满意度 | 理赔客户的满意度评分 | 不低于基线 |
| 流失率 | 客户流失比例 | 不超过基线的110% |
| 欺诈风险 | 理赔欺诈概率 | 不超过基线 |

## 4. 回滚条件

| 触发情形 | 动作 |
|----------|------|
| 连续3天出现显著SRM/分流漂移 | 暂停读数并排查 |
| 投诉率或理赔满意度显著恶化 | 停止扩量并回到上一版本 |
| 续保率或二开率转负 | 停止扩量 |
| Week 1抬升但Week 4回落 | 回滚并重做方案 |

## 5. 实验监控

### 5.1 每日监控
- 分流比例检查
- 主指标趋势
- 护栏指标监控

### 5.2 周度报告
- 实验效果汇总
- 统计显著性检验
- 异常情况说明

## 6. 扩量策略

### 6.1 扩量条件
- 主指标为正且统计显著
- 关键护栏指标稳定
- 实验周期达到30天

### 6.2 扩量档位
- 第一档：10%用户
- 第二档：30%用户
- 第三档：50%用户
- 第四档：100%用户
